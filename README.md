# wearther_frontend
2024-2 SOLUX Project

## 🌤️ Main 기능
- 회원가입/로그인: 사용자 위치·체질·취향·보유한 옷 설정
- 홈: 오늘의 추천템, 즐겨찾기한 위치의 실시간 날씨와 시간별 예보 확인
- 위치: 현위치 확인, 즐겨찾기할 위치 추가 및 삭제
- 옷장: 보유한 옷과 보유하지 않은 옷(추천 상품) 조회, 추천 상품 구매 사이트로 연결
- 마이페이지: 프로필 편집, 로그아웃, 계정 삭제, 푸쉬 알림 여부 설정
